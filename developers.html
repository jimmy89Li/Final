Create Apps In A RESTful Way

An API to the NorthQ Service Infrastructure is available to enable certified third party developers to create apps, websites or extend the capabilities of their own system. The NorthQ HomeManager API is a RESTful API and uses JSON as the exchange format. It uses HTTPS protocol for encrypted communication. SSL is mandatory and used for all communication.

Easy To Use, Meant For Everybody

The HomeManager API is very simple to use. Developers just need to focus on their own app and let the infrastructure behind the API take care of the rest. There is no need to be a professional programmer to get a prototype up and running.

The system provides opportunities for:

  • Controlling devices remotely;
  • Monitoring devices;
  • Tracking consumption;
  • Getting notifications from security devices;
  • Getting notifications from security devices;
  • Scheduling devices and other unlimited opportunities.
